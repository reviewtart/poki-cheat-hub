How to use

  1. 1 · Open the game on poki.com

    Wait until you can press play. The game frame loads on a subdomain like 5dd312fa-...gdn.poki.com — that's where the engine lives.

  2. 2 · Open DevTools → switch context

    Press F12 (or ⌘⌥I). In the Console's context dropdown, pick the game frame (it ends with gdn.poki.com). This is the cross-origin frame — your console must be inside it.

  3. 3 · Paste the snippet

    Hit a "Copy" button on this page, paste into the console, press Enter. Done.

  4. 3a · Or use the bookmarklet

    If you generated a bookmarklet, drag it to your bookmarks bar. Click it while the game tab is focused. Note: it must be invoked while the game frame is the active one — for many games that means clicking the canvas first.

  5. 3b · Or use the Tampermonkey userscript

    Install Tampermonkey, create a new script, paste the generated content. It auto-runs every time you load a Poki game.

FAQ

Why can't I just iframe Poki here and inject cheats?

poki.com uses a frame-ancestors CSP that blocks everything except *.poki.io and localhost. The actual game subdomain (*.gdn.poki.com) sends X-Frame-Options: SAMEORIGIN. So a hosted site can't legally frame the game. The bookmarklet / console / userscript flow runs in your own browser, in the game's own context, where same-origin rules don't get in the way.

Is this against Poki's TOS?

Modifying client-side JavaScript of a game running in your own browser is what game modding has always been. Poki's leaderboards are server-validated for many games — so submitting fake scores usually fails server-side. Use it for fun in offline play. Don't expect cheated scores to land on the global high-score table.

Will Poki detect and ban me?

Poki does not have user accounts you can be banned from for most games. The SDK collects analytics events; overriding them locally is a client choice. Don't run aggressive ad-blocking on revenue-sensitive accounts (creators).

Does this work on mobile?

Mobile browsers don't expose DevTools the same way. You can sometimes use Eruda or a remote-debug bridge. Bookmarklets work on iOS Safari and Chrome desktop.

How are new games added?

The hooks live in js/cheats/*.js as plain ES modules. Each one exports metadata plus snippet generators. Pull requests welcome — see the README in the repo.
